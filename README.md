# Ambient Temperature and Population-Level Medication Dispensing in Australia

Replication code and materials for:

> Farquhar H. Ambient Temperature and Population-Level Medication Dispensing in Australia: A Distributed Lag Non-Linear Analysis with Equity Stratification. 2026.

## Overview

This study examines the relationship between ambient temperature and pharmaceutical dispensing across Australia using distributed lag non-linear models (DLNMs). It analyses weekly Pharmaceutical Benefits Scheme (PBS) dispensing data (2012–2023) at the SA4 geographic level for cardiovascular (ATC C), mental health (ATC N05, N06), and respiratory (ATC R03) medications. A secondary difference-in-differences analysis examines the 2019–20 Black Summer bushfire season's impact on respiratory medication dispensing.

## Data Access

Raw data must be obtained separately due to size and licensing:

### PBS Dispensing Data
- **Source:** Australian Institute of Health and Welfare (AIHW) PBS data exports
- **URL:** https://www.aihw.gov.au/reports/pharmaceuticals/pharmaceutical-benefits-scheme-data
- **Files needed:** Monthly PBS dispensing counts by ATC code and geographic area
- **Place in:** `data/raw/pbs/`

### Climate Data (BOM AGCD)
- **Source:** Bureau of Meteorology Australian Gridded Climate Data (~5 km resolution)
- **Access:** Via Copernicus Climate Data Store (CDS) API — requires free registration
- **Variables:** Daily maximum temperature (Tmax), minimum temperature (Tmin), rainfall
- **Place in:** `data/raw/climate/`
- **Note:** Configure CDS API credentials following https://cds.climate.copernicus.eu/how-to-api

### Air Quality Data (OpenAQ)
- **Source:** OpenAQ API (free, requires registration)
- **URL:** https://openaq.org
- **Variable:** PM2.5 station-level measurements
- **Setup:** Copy `.env.template` to `.env` and add your API key
