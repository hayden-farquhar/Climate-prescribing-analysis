# Ambient Temperature and Population-Level Medication Dispensing in Australia

Replication code for:

> Farquhar H. Ambient Temperature and Population-Level Medication Dispensing in Australia: A Distributed Lag Non-Linear Analysis with Equity Stratification. 2026.

## Overview

This study examines the relationship between ambient temperature and pharmaceutical dispensing across Australia using distributed lag non-linear models (DLNMs). It analyses weekly Pharmaceutical Benefits Scheme (PBS) dispensing data (2012тАУ2023) at the SA4 geographic level for cardiovascular (ATC C), mental health (ATC N05, N06), and respiratory (ATC R03) medications. A secondary difference-in-differences analysis examines the 2019тАУ20 Black Summer bushfire season's impact on respiratory medication dispensing.

### PBS Dispensing Data
- **Source:** Australian Institute of Health and Welfare (AIHW) PBS data exports
- **URL:** https://www.aihw.gov.au/reports/pharmaceuticals/pharmaceutical-benefits-scheme-data
- **Files needed:** Monthly PBS dispensing counts by ATC code and geographic area
- **Place in:** `data/raw/pbs/`

### Climate Data (BOM AGCD)
- **Source:** Bureau of Meteorology Australian Gridded Climate Data (~5 km resolution)
- **Access:** Via Copernicus Climate Data Store (CDS) API тАФ requires free registration
- **Variables:** Daily maximum temperature (Tmax), minimum temperature (Tmin), rainfall
- **Place in:** `data/raw/climate/`
- **Note:** Configure CDS API credentials following https://cds.climate.copernicus.eu/how-to-api

### Air Quality Data (OpenAQ)
- **Source:** OpenAQ API (free, requires registration)
- **URL:** https://openaq.org
- **Variable:** PM2.5 station-level measurements
- **Setup:** Copy `.env.template` to `.env` and add your API key

### Reference Data
- **SEIFA 2021:** ABS Socio-Economic Indexes for Areas тАФ https://www.abs.gov.au/statistics/people/people-and-communities/socio-economic-indexes-areas-seifa-australia/latest-release
- **ERP:** ABS Estimated Resident Population by LGA
- **Remoteness:** ABS Remoteness Areas classification
- **Place in:** `data/reference/`

### Spatial Boundaries
- **Source:** ABS Australian Statistical Geography Standard (ASGS) 2021
- **Boundaries needed:** LGA, SA4, and Remoteness Area shapefiles
- **URL:** https://www.abs.gov.au/statistics/standards/australian-statistical-geography-standard-asgs-edition-3/jul2021-jun2026/access-and-downloads/digital-boundary-files
- **Place in:** `data/spatial/`
